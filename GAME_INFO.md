# 老人斗地主游戏说明\r\n\r\n## 项目概述\r\n这是一个专为老人设计的单机版斗地主游戏，具有以下特点：\r\n- 纯单机游戏，无需网络连接\r\n- 界面简洁，字体大，操作简单\r\n- 无任何广告、内购或套路\r\n- 专为老人优化的游戏体验\r\n\r\n## 主要功能\r\n1. 经典斗地主玩法\r\n2. 三个玩家（用户+两个电脑AI）\r\n3. 完整的牌型判断系统\r\n4. 智能提示功能\r\n5. 游戏记录查看\r\n6. 游戏规则说明\r\n\r\n## UI/UX 设计特点\r\n- 大字体设计，保护老人视力\r\n- 高对比度颜色，便于识别\r\n- 简洁的界面布局\r\n- 直观的操作方式\r\n- 长按功能快速选择相同牌\r\n- 提示功能帮助决策\r\n\r\n## 技术实现\r\n- 使用 React Native + Expo 框架\r\n- 完全离线运行，无网络请求\r\n- 无第三方广告SDK\r\n- 无应用内购买功能\r\n\r\n## 使用说明\r\n1. 点击"开始游戏"进入游戏\r\n2. 点击牌张可选中/取消选中\r\n3. 长按相同点数的牌可快速选中\r\n4. 点击"提示"按钮可获得出牌建议\r\n5. 点击"过牌"跳过当前回合\r\n6. 点击"出牌"打出选中的牌\r\n\r\n## 游戏规则\r\n- 游戏人数：3人（您和2个电脑玩家）\r\n- 牌数：每人17张，底牌3张\r\n- 出牌规则：按逆时针方向轮流出牌\r\n- 牌型大小：单张、对子、三张、顺子、连对、三带一、三带二、四带二、炸弹、王炸\r\n- 王炸最大，然后是普通炸弹\r\n- 先出完牌的玩家获胜\r\n\r\n## 安全性保证\r\n- 无任何广告\r\n- 无内购项目\r\n- 无数据收集\r\n- 完全离线运行\r\n- 无网络权限需求\r\n\r\n## 无障碍设计\r\n- 大按钮设计，便于点击\r\n- 高对比度配色\r\n- 清晰的视觉反馈\r\n- 简单直观的操作流程\r\n\r\n这个应用完全符合老人使用习惯，提供了一个纯净、简单的斗地主游戏体验。
