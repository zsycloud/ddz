# 老人斗地主

一款专为老人设计的单机版斗地主游戏，无广告、无内购、无套路，纯粹娱乐。

## 特点

- 纯单机游戏，无需网络
- 界面简洁，操作简单
- 大字体设计，保护视力
- 无任何广告和收费项目
- 专为老人优化的游戏体验
- 包含完整的叫地主功能

## 环境要求

1. **Node.js** (推荐 LTS 版本)
   - 从 https://nodejs.org/ 下载并安装

2. **Git**
   - 从 https://git-scm.com/ 下载并安装

3. **VSCode 扩展**
   - React Native Tools
   - Expo Tools
   - ES7+ React/Redux/React-Native snippets
   - Prettier - Code formatter

## 项目设置

1. 安装 Expo CLI:
   ```bash
   npm install -g @expo/cli
   ```

2. 安装项目依赖:
   ```bash
   npm install
   ```

## 运行项目

1. 启动开发服务器:
   ```bash
   npm start
   ```

2. 在 iOS 设备上测试:
   - 在 iOS 设备上安装 "Expo Go" 应用
   - 使用设备扫描终端中显示的二维码

## 游戏特色

- 简单易懂的操作界面
- 适合老人的大按钮和大字体
- 流畅的游戏体验
- 经典斗地主玩法
- 完整的叫地主机制（叫1分、2分、3分或不叫）
- 智能AI对手

## 游戏规则

### 发牌阶段
- 一副牌（54张）分给三人，每人17张
- 剩下3张作为底牌

### 叫地主阶段
- 按顺序（玩家→AI1→AI2）依次叫地主
- 可选择不叫（pass）或叫1分、2分、3分
- 叫分最高的玩家成为地主
- 如果都选择不叫，则重新发牌
- 地主获得3张底牌

### 游戏阶段
- 地主先出牌
- 按逆时针方向轮流出牌
- 农民（非地主玩家）合作对抗地主
- 先出完牌的玩家获胜
  - 如果地主先出完牌，则地主获胜
  - 如果任一农民先出完牌，则农民获胜

### 牌型
- 单张、对子、三张、三带一、三带二
- 顺子（五张或更多连续单张）
- 连对（三对或更多连续对子）
- 四带二（四张相同+两张单牌或两对）
- 炸弹（四张相同）
- 王炸（双王，最大牌型）
